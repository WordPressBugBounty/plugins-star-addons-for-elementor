A collection of multiple addons for elementor website builder plugin. It also helps you to create template kits for elementor.

== Description ==

Star Addons has an easy-to-use interface options for creating and customizing addons for ultimate needs. It is compatible with Elementor Website Builder, WooCommerce, LearnPress, Contact Form 7, WP Form, Ninja Form etc.

== Installation ==

= From the dashboard: =
1. Go to the WordPress Dashboard "Add New Plugin" section
2. Search For "Star Addons for Elementor"
3. Install, then Activate it

= From the control panel: =
1. Unzip (if it is zipped) and Upload `star-addons-for-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can I use the plugin without Elementor Website Builder? =

No. You cannot use without Elementor since it's an addon for Elementor.

= Does it work with any theme? =

Absolutely! It will work with any theme where Elementor works.
